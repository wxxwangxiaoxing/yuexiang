## 接口 1：获取个人中心首页数据

**用途**：页面加载时获取用户概要信息

```plain
GET /api/user/profile
```

### 请求头

| Header          | 必填 | 说明           |
| --------------- | ---- | -------------- |
| `Authorization` | 是   | Bearer {token} |

### 响应

```json
{
  "code": 200,
  "msg": "ok",
  "data": {
    "userId": 888888,
    "userIdDisplay": "HM_888888",
    "nickName": "小红薯",
    "avatar": "/imgs/avatar/user888888.jpg",
    "gender": 1,
    "level": 3,
    "levelName": "黄金会员",
    "levelIcon": "/imgs/level/gold.png",
    "points": 2580,
    "followCount": 128,
    "fansCount": 256,
    "likeCount": 1200,
    "isSignedToday": false,
    "continuousSignDays": 5,
    "unreadMessageCount": 3,
    "unpaidOrderCount": 2,
    "unusedVoucherCount": 5
  }
}
```

### 字段说明

| 字段                 | 类型    | 说明                     |
| -------------------- | ------- | ------------------------ |
| `userId`             | Long    | 用户ID                   |
| `userIdDisplay`      | String  | 展示用ID（带前缀格式化） |
| `nickName`           | String  | 昵称                     |
| `avatar`             | String  | 头像路径                 |
| `gender`             | Integer | 性别：0未知 1男 2女      |
| `level`              | Integer | 会员等级（1~10）         |
| `levelName`          | String  | 等级名称                 |
| `levelIcon`          | String  | 等级图标                 |
| `points`             | Integer | 当前积分                 |
| `followCount`        | Integer | 关注数                   |
| `fansCount`          | Integer | 粉丝数                   |
| `likeCount`          | Integer | 获赞总数（笔记被点赞数） |
| `isSignedToday`      | Boolean | 今日是否已签到           |
| `continuousSignDays` | Integer | 连续签到天数             |
| `unreadMessageCount` | Integer | 未读消息数（红点提示）   |
| `unpaidOrderCount`   | Integer | 待支付订单数             |
| `unusedVoucherCount` | Integer | 未使用优惠券数           |

### 数据来源

```sql
-- 基本信息
SELECT u.id, u.nick_name, u.avatar,
       ui.gender, ui.level, ui.points,
       ui.follow_count, ui.fans_count
FROM tb_user u
    LEFT JOIN tb_user_info ui ON ui.user_id = u.id
WHERE u.id = #{userId} AND u.deleted = 0;

-- 等级信息
SELECT name, icon FROM tb_member_level WHERE level = #{userLevel};

-- 获赞总数（笔记被点赞）
SELECT COALESCE(SUM(like_count), 0)
FROM tb_blog
WHERE user_id = #{userId} AND deleted = 0;

-- 今日签到状态（位图查询）
SELECT sign_bitmap FROM tb_sign
WHERE user_id = #{userId} AND year = #{year} AND month = #{month};
-- 判断: (sign_bitmap >> (day - 1)) & 1 == 1

-- 未读消息数
SELECT COUNT(1) FROM tb_message
WHERE to_user_id = #{userId} AND is_read = 0 AND deleted = 0;

-- 待支付订单数
SELECT COUNT(1) FROM tb_voucher_order
WHERE user_id = #{userId} AND status = 0 AND deleted = 0;

-- 未使用优惠券数
SELECT COUNT(1) FROM tb_voucher_order
WHERE user_id = #{userId} AND status = 1 AND deleted = 0;
```

```java
// 从 tb_sign.sign_bitmap 解析已签到日期
public List<Integer> parseSignedDays(int signBitmap) {
    List<Integer> days = new ArrayList<>();
    for (int i = 1; i <= 31; i++) {
        if ((signBitmap >> (i - 1) & 1) == 1) {
            days.add(i);
        }
    }
    return days;
}

// 判断某天是否签到
boolean isSigned = (signBitmap >> (day - 1) & 1) == 1;
```

```java
@Transactional
public SignResult sign(Long userId) {
    LocalDate today = LocalDate.now();
    int year = today.getYear();
    int month = today.getMonthValue();
    int day = today.getDayOfMonth();
    
    // 1. 查询或创建当月签到记录
    TbSign sign = signMapper.selectByUserMonth(userId, year, month);
    if (sign == null) {
        sign = new TbSign();
        sign.setUserId(userId);
        sign.setYear(year);
        sign.setMonth(month);
        sign.setSignBitmap(0);
        sign.setSignDays(0);
        signMapper.insert(sign);
    }
    
    // 2. 判断今天是否已签到
    int bitmap = sign.getSignBitmap();
    if ((bitmap >> (day - 1) & 1) == 1) {
        throw new BizException(40020, "今日已签到");
    }
    
    // 3. 计算连续签到天数
    int continuousDays = calculateContinuousDays(userId, today);
    
    // 4. 计算奖励积分
    int points = calculatePoints(continuousDays + 1);
    
    // 5. 更新签到位图
    int newBitmap = bitmap | (1 << (day - 1));
    signMapper.updateBitmap(sign.getId(), newBitmap, sign.getSignDays() + 1);
    
    // 6. 增加积分
    userInfoMapper.addPoints(userId, points);
    
    // 7. 记录积分流水
    pointsRecordMapper.insert(new TbPointsRecord(
        userId, 1, points, "签到奖励", "SIGN", sign.getId()
    ));
    
    return new SignResult(today, points, continuousDays + 1);
}
```

### 连续签到计算

```java
private int calculateContinuousDays(Long userId, LocalDate today) {
    int continuous = 0;
    LocalDate checkDate = today.minusDays(1);
    
    while (true) {
        int year = checkDate.getYear();
        int month = checkDate.getMonthValue();
        int day = checkDate.getDayOfMonth();
        
        TbSign sign = signMapper.selectByUserMonth(userId, year, month);
        if (sign == null) break;
        
        if ((sign.getSignBitmap() >> (day - 1) & 1) == 1) {
            continuous++;
            checkDate = checkDate.minusDays(1);
        } else {
            break;
        }
    }
    return continuous;
}
```

### 积分奖励规则

```java
private int calculatePoints(int continuousDays) {
    // 从 tb_sys_config 或配置中心读取
    if (continuousDays >= 7) return 50;  // 连续7天
    if (continuousDays >= 5) return 20;  // 连续5-6天
    if (continuousDays >= 3) return 15;  // 连续3-4天
    return 10;                            // 1-2天
}
```

## 数据权限与安全

| 安全点   | 实现方式                                                     |
| -------- | ------------------------------------------------------------ |
| 数据隔离 | 所有接口仅查询当前登录用户数据，`WHERE user_id = #{currentUserId}` |
| 登录校验 | 所有接口均需 Token 认证                                      |
| 分页限制 | `pageSize` 最大值限制为 50，防止拉取过多数据                 |
| 敏感信息 | 不返回密码、身份证等敏感字段                                 |
| 操作审计 | 删除足迹、签到等操作记录日志                                 |
